特務(Agent)

在《特戰英豪》中，玩家所控制的角色稱為特務，在比賽開始前需選擇一名特務，鎖定選擇後皆不得更換。每位特務均有專屬的技能，可在回合開始前於商店選單中進行購買。

  • Let's Go

模式(Mode)

有一般模式、競技模式、炸彈搶攻戰、死鬥模式、自訂模式等五種模式，讓玩家遊玩

  • Learn More

武器(Weapon)

武器有分隨身武器、衝鋒槍、霰彈槍、步槍、狙擊槍、重型武器、近戰武器，開場前可讓玩家依金錢多寡選擇購買

  • Learn More

地圖(Map)

遊戲總共五張地圖，分別為「劫境之地」、「遺落境地」、「雙塔迷城」、「義境空島」以及「極地寒港」

  • Learn More

特務(Agent)

遊戲內的特務分別來自世界各國，他們各自擁有獨特的文化及專長。目前遊戲中共計十三位特務，至今仍持續不斷更新中

  • Watch video

婕提(Jett)

身為南韓代表，婕提身手矯捷，能使出他人所不能的精妙險招。交戰時她能把所有人耍得團團轉，以迅雷不及掩耳的速度秒殺敵人

  • Details

瑞茲(Raze)

來自巴西的芮茲經常使用巨型槍械展現她大喇喇的個性。她擅長用驚天一「砰」，逼出躲在壕溝的埋伏，掃除狹窄空間的敵人，這就是她的暴力破壞風

  • Details

叛奇(Breach)

叛奇是瑞典的生化人，他能發射強大且精準的雷射震擊，通過火力壓制開闢前往敵區的道路。能夠阻斷敵方，又能造成傷害的他，絕對是確保敵我雙方實力懸殊的不二人選

  • Details

歐門(Omen)

歐門是個記憶中的魅影，在暗影裡巡獵。他能蒙蔽敵人雙眼、在地圖上神出鬼沒，使詭譎的氣氛籠罩全域，而他的敵人永遠在心慌意亂之中，試圖猜測他接下來會在哪裡現身

  • Details

布史東(Brimstone)

來自美國的布史東擁有強大的衛星武器，讓他的隊伍永遠處於優勢。他能投放戰略工具支援前線，包準「恰到好處」且萬無一失，讓他成為地表最強指揮官

  • Details

菲尼克斯(Phoenix)

菲尼克斯來自英國，他的星辰之力融會於其戰鬥風格中，其閃光彈和照明彈足以讓星火燎原。不管有沒有後援，他都能衝鋒陷陣，獨自撂倒敵人

  • Details

聖祈(Sage)

來自中國的聖祈，是宛如銅牆鐵壁一般的存在，無論身在何處，除了自保之外，也能確保隊伍的安全。她可以復活不支倒地的友軍，屏除猛烈的攻勢，在槍林彈雨的戰鬥中，建造一座令人安心的庇護所

  • Details

蘇法(Sova)

蘇法出生於俄羅斯的永凍冰原，他靠著快捷而精準的身手，緊追在後、曝光敵蹤、將目標無情殲滅。他手持改造弓，結合出色的偵查能力，讓你逃得了一時，躲不過一世

  • Details

薇蝮(Viper)

薇蝮是名美國化學家，她的特長是部署五花八門的劇毒化武，掌控戰局的同時，還能妨礙敵人視野。即使毒藥沒能殺死對手，她的心理戰術也會虐死人

  • Details

瑟符(Cypher)

瑟符是摩洛哥的情報販子，他隻身一人便是頂尖情報中心，窺探著敵人的一舉一動。沒有計謀可以逃過他的法眼。瑟符永遠監視著你

  • Details

蕾娜(Reyna)

在墨西哥中心經過鍛鍊的蕾娜，有強大的單挑能力，每擊殺一名敵軍，都可以獲得所向無敵的增益效果。她是否能主宰戰場，除了取決於技巧熟練度，也仰賴賽局上的表現

  • Details

愷宙(Killjoy)

愷宙是位來自德國的天才，總是氣定神閒，靠著她火力強大的發明來鎮守戰場。她的機器人能對戰局造成十足的影響力，就算沒把敵人立即擊倒，也足以靠著附加的減益效果使他們舉步維艱

  • Details

絲凱(Skye)

絲凱來自澳洲，她與她的野獸同伴會勇闖敵區，為我方開路。除了能召喚生物妨礙敵軍，她還能治療隊友，因此只要有絲凱在側，肯定能展現隊伍的最強實力，避開凶險局面

  • Details
